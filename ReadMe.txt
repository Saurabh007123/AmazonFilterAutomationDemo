Selenium Automation Script are included in the Projectfolder "TrialArtifact"

It is build on Maven Project including TestNG Framework
In the project:
(1) TestNG.xml file have methods are maintained in a certain execution order
(2) Locator.properties file have all the element locators and local configuration like Browser, URL etc
(3) While test case execution Product details like Price, Rating, Title and Review are get stored in DataStore.csv file which creates and save a new csv file with date and time stamp with new run


Recording is in the project folder with mp4 format
